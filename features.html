Platform Capabilities

Everything Your Classroom Needs

Six powerful, AI-driven capabilities working in harmony — so you can focus on teaching while the system handles the data.

Real-Time Emotion Detection

Computer vision models continuously analyse facial micro-expressions to classify student states: focused, confused, bored, happy, or disengaged — with per-frame updates every 500ms.

OpenCV FER+ DeepFace WebRTC
View documentation

Speech & Participation Analysis

Directional microphones track who is speaking, for how long, and with what confidence. NLP sentiment analysis scores verbal contributions — helping identify silent students who need encouragement.

Whisper ASR pyAudioAnalysis BERT NLP
View documentation

Cloud-Based Processing

Edge devices handle local inference for minimal latency. Aggregated, anonymised metrics sync to a secure cloud for long-term trend analysis, cross-classroom benchmarking, and school-wide reporting.

AWS IoT MQTT TensorFlow Lite Kafka
View documentation

Teacher Dashboard

A clean, intuitive web dashboard gives teachers a live view of class engagement. Drill down into individual student timelines, export PDF reports, and receive smart alerts when attention drops below threshold.

React Chart.js WebSocket D3.js
Try live demo

Natural Language Query Interface

Teachers ask plain English questions — "Who was most distracted during Period 3?" — and the system returns structured, visual reports instantly. No SQL, no filters, no training required.

GPT-4o LangChain RAG Semantic Search
Try it now

Privacy-Compliant Data Handling

No raw video or audio ever leaves the classroom. All biometric data is processed on-device and immediately discarded. Aggregated metrics are anonymised, encrypted at rest and in transit — fully FERPA & GDPR compliant.

On-Device AI AES-256 FERPA GDPR
